조건문이란 ? 
 - 조건이 true 일 때만 지정한 영역을 실행하도록 하는 명령문
 - 특정 영역을 수행할 지 여부를 결정짓기 때문에, 결국 실행부에 대한 제어 역할을 하므로 제어문중 하나임.

유형
 - if (단일 조건)
 - if ~ else (조건 판단 대상 딱 2가지)
 - else ~ if (조건 판단 대상 여러개)
 - switch (else ~ if 문을 간략히 작성할 때)
            (단, switch 문의 조건 대상은 고정된 값(상수, 숫자 문자 논리값)인 경우에만 적절.)
            (따라서 다양한 조건식의 표현일 경우 else~if가 적합)

--------------------------------------------------------------------------------------------------------

if ~ else  vs  삼항연산자

if ~ else 의 경우가 훨씬 다채로움. (구조화된 로직을 표현하기 쉬움)
중첩 조건의 표현시 필요함
명령문을 자유롭게 넣을 수 있음

삼항 연산자는 코드가 간결하고 가독성이 있음.

반복문 시작

반복문이란 ? 
 - 지정된 영역을 지정된 반복 횟수 또는 조건이 true 동안 수행하도록 하는 명령문
 - 특정 영역을 얼만큼 반복할 지 여부를 결정짓기 때문에 , 결국 실행부에 대한 제어 역할, 따라서 제어문

--------------------------------------------------------------------------------------------------------

반복문의 유형

1. for  - 지정된 횟수동안 실행되는 반복문

for (let 변수=1; 변수 =< 반복시킬횟수 ; 변수++) {
}

--------------------------------------------------------------------------------------------------------

2. while - 조건이 true 동안 계속 실행되는 반복문




--------------------------------------------------------------------------------------------------------

3. do... whilte - 조건과 상관없이 최소 1번은 무조건 실행되는 반복문



--------------------------------------------------------------------------------------------------------

4. for ... in - 객체가 보유한 key의 수 만큼 실행되는 반복문 (배열에서 사용 권장x) 반복하면서 key 변환




--------------------------------------------------------------------------------------------------------

5. for ... of - 배열, 문자열, 이터러블한 객체의 요소 수 만큼 실행되는 반복문

- 사원의 급여의 합

SELECT SUM(SAL) FROM EMP;

- 사원의 커미션의 합 

SELECT SUM(COMM) FROM EMP;

- 사원의 급여의 평균
SELECT AVG(SAL) FROM EMP;

- 컬럼명을 바꿔 출력하고 싶을 때
SELECT AVG(SAL) AS 급여합 FROM EMP;

- DEPTNO 끼리 묶어서 출력해줘 (GROUP BY)
SELECT DEPTNO FROM EMP GROUP BY DEPTNO;

카운트 한 뒤 묶어서 출력해줘
SELECT DEPTNO, COUNT(DEPTNO) FROM EMP GROUP BY DEPTNO;
