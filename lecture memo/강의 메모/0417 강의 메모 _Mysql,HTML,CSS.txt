단어 정리 

HTML = Hyper Text Markup Language = 링크가 지원되는 마크업 기반 언어
Web Browser = HTML을 해석할 수 있는 프로그램
Web Server = HTML(문서)를 저장 후 Web Browser 로 보내줌.
HTTP Protocol = 웹서버와 웹브라우저간의 사전약속 (전송규약)
Web = HTML을 보유하고있는 서버끼리 연결되어있는 형상 
웹 퍼블리셔 = HTML을 전문적으로 하는사람

3교시. CSS 수업 시작
--------------------------------------------------------------------------
1. 인라인 CSS (인라인)

 - HTML 요소의 style 속성을 이용하며 직접 스타일 적용
 스타일 시트로 적은 경우 , 모든 걸 다 바꿔야 하므로 불편함. (반복적이지 않은 업무에만 적용)
 - <td style="font-size:9px"> 


2. 내부 CSS (인터널)

- <style> 태그를 사용하여 html 파일 내부의 스타일 지정
- 문서량이 적은 소규모에 적당 , 여러 페이지에서의 재사용성은 떨어짐
<style>
	td{
		font-size:9px;
		color:"red"
	}
</style>

위 1번과 2번은 html 파일에 직접 적지만 , 3번의 외부는 파일을 적용. (내부에 적지 않음)
--------------------------------------------------------------------------
3. 외부 css 

 - css파일을 만들어 HTML 문서와 분리하여 스타일 적용
 - 재사용성이 높고 유지 보수가 편리함.

파일을 새로 만들되 , 확장자명이 중요 (파일명.css)
--------------------------------------------------------------------------
css 구문 예시
<style>
p { //p 태그에 대해
 	color: red; //폰트 색상 red
	font-size: 20px // 폰트 크기 20px
}

	width:100px; height:50px;
}
</style>
