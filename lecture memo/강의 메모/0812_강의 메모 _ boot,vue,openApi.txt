HTML 과 XML 의 차이.

HTML : 어떻게 보여줄지 (CSS , Vue 등등)
	HTML 은 프로그래밍 언어가 아니므로 , JS로 제어가 불가능함.


XML : 데이터를 어떻게 표현할지 (text기반의 데이터베이스. 구조화 되어있음.)

--------------------------------------------------------------------------------------------------------

XML 파싱 
	- DOM(Document Object Model) 방식 : 파싱 쉽고 처리가 간단함. 
							단 , 메모리에 모든 요소를 DOM으로 올리는 방식이라 메모리 자원을 많이 사용함.
							따라서 스마트폰용 웹을 동시에 지원하는 프로젝트일 경우 사용 지양함.

	- SAX 방식 : 실행부가 XML 문서를 위에서 아래 방향으로 진행하면서 , 적절한 이벤트를 일으켜
			해당 이벤트와 관련된 메서드를 호출하는 방식.
			개발자는 이벤트 발생 시 적절한 타이밍을 놓치면 안됨.
